Developed RestroTrack, a mobile based inventory management application tailored for the restaurant industry. The app streamlines inventory tracking, minimizes food waste, and automates stock control using real-time updates, barcode scanning, expiry alerts, and supplier coordination. Built with Kotlin and Jetpack Compose, and integrated with Firebase for cloud-based data management, the system supports offline functionality, role based access, and POS synchronization. Designed to be scalable, user-friendly, and cost effective, RestroTrack enhances operational efficiency and supports sustainability practices across restaurant environments.

Skills: Kotlin, Android Jetpack
